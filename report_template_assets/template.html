Compares total time by phase across the loaded wells. Actions with code D, DMR, and DMS are excluded from this view.

Shows only the drilling actions D, DMR, and DMS by phase. Drilling footage will be added automatically when the source file includes depth or footage fields.

Depth vs Days Progression

Shows cumulative days from the initial to the final operation. When true depth is not available in the flat time export, the Y axis uses ordered phase progression as a depth proxy.
